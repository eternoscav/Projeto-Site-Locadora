DICAS & TRUQUES
Meta bala em Hitler!!
Selecione "Game Options" no menu principal, então escolha a opção para habilitar o console. Pressione ( ' ) para baixar o console e entre com o comando developer 1 para habilitar as dicas. O botão "Load" irá aparecer. Clique sobre ele e selecione o nível desejado. Depois que o nível começar, pressione ( ' ) para baixar outra vez o console e então entre com o comando devmap. Pressione ( ' ) durante o jogo para baixar mais uma vez o console e entre com um dos seguintes códigos abaixo.

Codigo Resultado
god Invencível
give all Duas armas diferentes
give ammo Munição
noclip Sem Clipping Mode
notarget Desliga a IA do inimigo

ATENÇÃO: OS CÓDIGOS PODEM NÃO FUNCIONAR EM DIFERENTES VERSÕES DO JOGO
